WEB CONSOLE

First you have to install visual studio and open it. After that in order to write code you must create a source code folder which will be used in your project. After creating, go to visual studio and open your exact same folder. Then, you have to create a file for your HTML. Then you can start to write the code.

In the console screen you can print anything using the "console.log" code and if you want to open the screen simply right click the page, click on inspect and select the console tab. Everything that you print using "console.log" is going to be shown here.
